# 📊 Mutual Fund Analytics Dashboard

An end-to-end Mutual Fund Analytics project developed using Python, PostgreSQL, SQL, and Power BI to analyze mutual fund performance, investor behavior, and SIP trends through interactive dashboards.

---

## 📌 Project Overview

This project demonstrates the complete Data Analytics workflow:

- Data Collection
- Data Cleaning using Python
- Data Transformation
- PostgreSQL Database Integration
- SQL Analysis
- Power BI Dashboard Development
- Business Insights Generation

---

## 🚀 Tech Stack

- Python (Pandas, NumPy, Jupyter Notebook)
- PostgreSQL
- SQL
